# AI-SDLC v0.7.11 Release Notes

## Positioning

`0.7.11` is an offline runtime packaging hardening patch on top of `0.7.10`.

It closes a production gap where release assets could pass CI while still
depending on the runner's local Python runtime. The macOS asset exposed the
problem first, but the fix treats Windows, macOS, and Linux as one release
contract.

## What Changed

### 1. Bundled runtime verification is now a release gate

Release build and published-asset smoke workflows now verify that each platform
bundle is marked with `python_runtime_bundled=true`, contains an executable
runtime, and that installation logs prove `Using bundled Python runtime`.

### 2. Runtime symlinks may not escape the bundle

Offline bundle construction now dereferences runtime symlinks while copying and
verifies that the resulting `python-runtime/` payload does not point outside the
archive. This prevents a bundle from working on a GitHub runner only because an
absolute host-toolcache path exists there.

### 3. Python wheel ABI is explicit

The bundle manifest now records the Python wheel ABI version used to download
dependencies. Installers reject an incompatible selected Python early instead of
letting pip fail later with missing binary wheels such as `pyyaml`.

## Compatibility Notes

- macOS / Linux use `.tar.gz` platform packages:
  - macOS Apple Silicon uses `ai-sdlc-offline-0.7.11-macos-arm64.tar.gz`
  - Linux x64 uses `ai-sdlc-offline-0.7.11-linux-amd64.tar.gz`
- Windows x64 uses `ai-sdlc-offline-0.7.11-windows-amd64.zip`
- Users stuck on older releases, including CLIs that report
  `No such command 'install'`, should download the matching package from
  `releases/download/v0.7.11` and run `--upgrade-existing` once.
- After `0.7.11` is installed, normal future updates should return to
  `ai-sdlc self-update check`.

## Verification

Required before marking `v0.7.11` complete:

- `uv run pytest tests/integration/test_offline_bundle_scripts.py tests/integration/test_github_workflows.py -q`
- `uv run pytest -q`
- `uv run ruff check src tests packaging/offline/verify_offline_bundle.py`
- `uv run ai-sdlc verify constraints`
- `uv build`
- `release-build.yml` for tag `v0.7.11`
- `release-artifact-smoke.yml` for tag `v0.7.11`
